# instant-layer
**Code generator for Microservices projects.**
![GitHub License](https://img.shields.io/github/license/Flaviogonzalez/instant-layer?style=for-the-badge)
![GitHub last commit](https://img.shields.io/github/last-commit/Flaviogonzalez/instant-layer?style=for-the-badge)
instant-layer is a Go code generator designed to quickly scaffold the base structure of microservices. (and even more in the future!)

By defining a service configuration in Go, instant-layer automatically generates the necessary files for an HTTP server, including database configuration, routing, and handler placeholders.
